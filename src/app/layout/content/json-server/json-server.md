## Json-Server

#### Step 1:
```
npm install -g json-server 
```

#### Step 2:
Create a _db.json_ file in a directory.

#### Step 3:
We need to run the json-server directory where db.json file is lying.
```
json-server --watch db.json 
```

#### Step 4:
```
http://localhost:3000
```
